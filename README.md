# advent-of-code

Welcome, this repository contains all of my solutions for AoC.

* For now all of them are done with TypeScript as that's the language I feel the most comfortable in.
* No AI tools were used in the process, I plan to stay away from them for the duration of the event (sorry ChatGPT, my beloved).
* Other than some basic tools for fetching input, solution checking and some helper functions for common operations needed for a lot of tasks there's really not much else going on here.

Good luck to everyone participating and let's have some fun 😁
